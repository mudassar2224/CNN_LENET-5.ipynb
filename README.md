# 👕 Clothing Classifier using AlexNet

A Deep Learning-based clothing image classification system built using TensorFlow and AlexNet architecture.

## 🚀 Project Overview

This project classifies clothing images into 14 categories using a custom AlexNet CNN model trained on a real-world fashion dataset.

The project includes:

* Data preprocessing
* Image augmentation
* CNN model training
* Model evaluation
* Gradio web interface
* Hugging Face deployment

---

## 📂 Dataset

Dataset used:
Clothing Dataset Full

Total Images: ~5000
Classes: 14 clothing categories

Examples:

* T-Shirt
* Pants
* Shoes
* Dress
* Hoodie
* Blazer
* Shorts
* Skirt

---

## 🧠 Model Architecture

Architecture Used:

* AlexNet-inspired CNN
* Batch Normalization
* Dropout Regularization
* Mixed Precision Training

Input Size:
224 × 224 × 3
